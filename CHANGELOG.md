# Changelog

## `v0.6.2` (2026-03-11)

- 전체 소스 코드를 src 디렉토리 기반으로 프로젝트 구조 변경

- 설정 시스템 전면 개편
  - 상수 데이터 단일화 (SSOT): 설정 상수와 타입을 하나로 통합하여 런타임 간 데이터 일관성 확보
  - 설정 동기화 오류 수정: 백그라운드와 설정 페이지 간의 실시간 동기화 이슈를 스토리지 이벤트 리스너를 통해 해결 (Race Condition 방지)

- 설정 UI/UX 고도화
  - 설정 항목의 그룹화 및 레이아웃 개선으로 접근성 향상
  - 설정 항목별 설명 문구를 더 직관적으로 수정하여 각 기능에 대한 이해도 증대

## `v0.6.1` (2026-03-04)

- 강의 목록 데이터 캐시 오류 수정
- 이클래스에서 읽어도 안사라지는 알림수 가리는 기능 추가

## `v0.6.0` (2025-10-09)

- Background 스크립트 레이어드 아키텍처 리팩토링 (Services, Managers, Repositories 레이어 분리)
- Repository 패턴을 통한 Chrome Storage 추상화 및 캐싱 시스템 개선
- CONTRIBUTING.md 문서 추가
- README.md 프로젝트 소개와 기여 부분 분리
- 탭 전환시 사이드패널이 자동으로 닫히는 기능 추가
- e-러닝 콘텐츠도 감지하여 VOD 목록에 추가
